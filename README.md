# LAB_TERMINAL //: REACT

LAB_TERMINAL is a modular laboratory environment for React experiments with a Python-backed telemetry layer. The current application combines a mechanical terminal-inspired React interface with a FastAPI backend that exposes project statistics, weather data, and research journal storage.

## Current Stack

The frontend uses React 19, Vite, React Router, Tailwind CSS v4, React Icons, Axios, and the Tailwind Vite plugin. The backend uses Python, FastAPI, Uvicorn, Pydantic, `httpx`, and `python-dotenv`.

## Current Application Surface

The primary shell lives in `src/app/AppLayout.jsx`, with routing centralized in `src/app/router.jsx`. Current pages are Dashboard, Active Tests, Research Journal, Telemetry Calibration, Documentation, and a Not Found route.

Dashboard shows system status, weather data, and backend project metadata from `/api/stats`. Active Tests provides local experimental module cards with file ingestion and scan simulation. Research Journal posts, reads, and deletes entries through the FastAPI journal endpoints backed by `backend/journal_vault.json`. Telemetry Calibration is a local-only control surface for future backend project telemetry, with tactile controls, derived meters, and an operational event log. Documentation is currently a UI route for curated resource links and can grow into an in-app reference surface.

## Backend API

The FastAPI app starts from `backend/main.py` on `127.0.0.1:8000`. It currently exposes root health text, weather lookup, project statistics, and journal CRUD-style routes. Project statistics are generated by `backend/backend_core/project_stats.py`, which returns totals for files, directories, total bytes, average file size, maximum directory depth, file type distribution, and the largest files.

## Theme Direction

All UI should continue to follow the LAB_TERMINAL mechanical laboratory aesthetic. Theme tokens are defined in `src/index.css`, including Safety Orange, Isotope Green, dark panel backgrounds, border colors, and glow effects. Prefer existing utility classes such as `.journal-panel`, `.journal-button`, `.journal-select`, and `.section-label` before creating new component styling.

## Getting Started

Install frontend dependencies with `npm install`, then install backend dependencies with `python -m pip install -r backend/requirements.txt`.

Run the frontend with `npm run dev`. Run the backend with `python backend/main.py`. The frontend expects the backend at `http://127.0.0.1:8000`.

Weather data requires a backend environment file with `WEATHERAPI_KEY` and `WEATHERAPI_ZIP`. Keep secrets out of Git.
