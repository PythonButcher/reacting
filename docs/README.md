# LAB_TERMINAL Documentation Index

This directory records the current project shape so future React and FastAPI work starts from a shared map.

## Documents

`PROJECT_OVERVIEW.md` explains the application purpose, route surface, and current feature modules.

`FRONTEND.md` documents the React 19 and Vite frontend architecture, theme rules, and component conventions.

`BACKEND.md` documents the FastAPI server, current endpoints, runtime data files, and environment requirements.

`DEVELOPMENT.md` documents setup, run commands, verification commands, and local workflow notes.

`GITIGNORE.md` explains why the repository ignores particular generated files, secrets, caches, and local data.

## Maintenance Rule

When routes, backend endpoints, runtime files, or theme tokens change, update the matching document in this directory in the same change set.
