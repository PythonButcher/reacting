# Visual System & Theme

Aesthetic: Highly skeuomorphic, retro-laboratory interface. Hardware aesthetic: precise labels, compact panels, restrained glow, mono typography, dark mechanical surfaces, tactile controls (dials, levers, knobs).
Tools: Tailwind CSS v4, custom CSS variables defined in src/index.css.
Key Tokens: --color-accent-primary (Safety Orange), --color-accent-secondary (Isotope Green), --color-bg-main, --color-bg-panel, --color-text-main, --color-text-dim.
AI Mandate: The AI assumes full responsibility for implementing and polishing complex CSS to match this hardware aesthetic.
